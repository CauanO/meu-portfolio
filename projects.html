Express - Rotas

Servidor HTTP utilizando o framework Express.js em Node.js. O código cria rotas para três URLs diferentes: "/", "/sobre" e "/contato". Cada rota responde enviando um arquivo HTML correspondente, localizado na pasta html. O servidor está configurado para escutar requisições na porta 8081 e exibir uma mensagem no console quando estiver funcionando.
